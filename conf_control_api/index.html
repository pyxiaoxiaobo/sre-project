鉴权

第一种方式 使用token
headers里 添加token 密钥请咨询管理员
token:方式 仅限会控相关接口

发起会议

url:/confapp/contact/create_conf/
方法:post
参数:

    data:{
        "attendee": [{"phone": "15268117291","name": "孙博",}],
        "theme": "",
        "password": "",
        "chairman": {"phone": "15268117291","name": "孙博",},
        "type": "1",
    }

attendee:与会人,[{"phone": "15268117291","name": "孙博",}] 【必填】
theme: 会议主题 默认是发起者的会议【选填】
password:密码【选填】
chairman:主席位 【必填】
type: 1 立即会议 2 预约会议【选填 默认是1】
time: 预约会议开始时间 格式为2020-03-03 19:30:36【选填 预约会议必填】

返回信息:
{
    "msg": "success",
    "code": 1,
    "data":"0570", 会议id
}

取消预约会议

url: /confapp/contact/cancel_meeting/
方法:post
参数:

confid:"0570" 会议id 【必填】<br>
chairman: "孙博" 操作人即主席只有主席可操作 【必填】<br>

返回信息:
{
    "msg": "success",
    "code": 1,
    "data":"0570"
}

编辑预约会议

url:/confapp/contact/edit_conf/
方法:post
参数:

    data:{
        "attendee": [{"phone": "15268117291","name": "孙博",}],
        "theme": "",
        "password": "",
        "chairman": {"phone": "15268117291","name": "孙博",},
        "type": "2",
    }

confid:会议id【必填】
attendee:与会人,[{"phone": "15268117291","name": "孙博",}] 【选填】
theme: 会议主题 默认是发起者的会议【选填】
password:密码【选填】
chairman: 主席位【必填】
type: 1 立即会议 2 预约会议【选填 默认是1】
time: 预约会议开始时间 格式为2020-03-03 19:30:36【选填 预约会议必填】

返回信息:
{
    "msg": "success",
    "code": 1,
    "data":"",
}

开麦闭麦

url:/conapp/control/mike/
方法:post
参数:

    data:{
        "confid":"0570",
        "attendee": [{"name":"张三", "phone":"15268117291"}]
        "type":"3",
        "chairman":"张三"
    }

confid:会议id【必填】
type: 3全部闭麦 4全部开麦 【必填】
chairman: 操作人即主席只有主席可操作【必填】
ex_chariman: True 除主席和发言人外【选填】
attendee:成员[{"name":"张三", "phone":"15268117291"}] 选传 指定人员开麦闭麦【选填】

返回信息:
{
    "msg": "success",
    "code": 1,
    "data": ""
}
