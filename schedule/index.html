运行

url:/schedule/run/

方法:POST

请求参数:
tool_name: 工具名
params: 参数

示例：

{
    "status": 1,
    "data": {
        "id": 17,
        "task_id": "f10e336c-bbb9-4643-9a18-2fdf29fc9a3f",
        "task_name": "21574384240",
        "status": "PENDING",
        "result": "",
        "create_time": "2019-11-22T00:57:20.850Z"
    },
    "msg": "调用成功"
}

查询运行状态

url:/schedule/run/

请求参数:
id: 任务id
