查询已被授权人员

url:/tools/authorizedUser/

方法:GET

请求参数:
tool_id: 工具id

示例：

{
    "status": 1,
    "data": {
        "核心网运行部": [
            {
                "华为数据组": [
                    "zz",
                    "sunbo222"
                ]
            },
            {
                "诺基亚组": [
                    "sunbo"
                ]
            }
        ],
        "监控部": [
            {
                "C&M小组": [
                    "lll"
                ]
            }
        ]
    },
    "msg": "查询成功"
}

查询已被授权人员列表

url:/tools/authorizedUser/list

方法:GET

请求参数:
tool_id: 工具id

为工具添加组的权限

url:/tools/authorityAdd/group/

方法:POST

请求参数:
tool_id: 工具id group_id: 组id 【多个组以';'号隔开】

为工具添加人的权限

url:/tools/authorityAdd/user/

方法:POST

请求参数:
tool_id: 工具id user_id: 人id 【多个人以';'号隔开】
