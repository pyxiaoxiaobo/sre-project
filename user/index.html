查询用户信息

url: users/userInfo/

方法：GET

参数: 用户id：id

返回示例：
is_active: 账号是否激活，False表示账号被停用
role: 账号角色；1管理员，2用户，3游客
username: 用户名
group: 组
depart: 部门
id: id
phone: 手机号

{
    "status": 1,
    "data": {
        "id": id,
        "username": "sunbo",
        "phone": "15268117291",
        "is_active": true,
        "role": 管理员,
        "group": "华为数据组",
        "depart": "核心网运行部"
    },
    "msg": "查询成功"
}

编辑账号信息

url: users/userInfo/

方法：PUT

参数: id: 被修改账号的id
username:被修改账号的用户名
password:被修改账号的密码
phone:被修改账号的手机号
group_id:被修改账号的组id
role:被修改账号的角色（1，2，3）
1, 管理员
2, 用户
3, 游客

删除账号

url: users/userInfo/

方法：DELETE

参数:
用户：id
需要删除的账号id

获取csrf_token

url:/users/csrftoken/
方法:GET

登录

url:/users/login/

方法:POST

请求参数:
用户名：username
密码：password

返回信息示例：

{
    "status": 1,
    "data": {
        "id": 1,
        "username": "sunbo",
        "session_id": "44ohfud9s6kkjfo3m5a3olr42mu5w2pp"，
        "phone": "8888888",
        "group": "test",
        "depart": "test",
        "role": "用户"
    },
    "msg": "登录成功"
}

退出登录

url:/users/loginout/

方法:GET

请求参数: 无

启用账号

url: users/active/
方法: POST
参数:
id: 被修改账号id
is_active: 1代表启用账号

停用账号

url: users/active/
方法: POST
参数:
id: 被修改账号id
is_active: 0代表停用账号

修改角色

url: users/role/
方法: POST
参数:
username:用户名
role:状态
1 管理员 2 用户 3 游客

用户注册

url:/users/register/

方法:POST

请求参数:
username:用户名
password:密码
role:角色
group_name:组名
phone_number:手机号

返回信息示例：

{
    "status": 1,
    "data": {
        "username": "sunbo3",
        "phone": "test",
        "role": "1"
    },
    "msg": "注册成功"
}

添加部门

url: users/department
方法: POST
参数:
部门名: name
示例

{
    "status": 1,
    "data": {
        "id": 6,
        "name": "test"
    },
    "msg": "添加成功"
}

id 部门id
name 部门名字

添加组

url: users/group
方法: POST
参数:
组名: group_name
部门id: depart_id
示例

{
    "status": 1,
    "data": {
        "group_name": "test",
        "depart_id": "6",
        "id": 8
    },
    "msg": "添加成功"
}
