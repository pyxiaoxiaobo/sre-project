1.工具上传

  • 1.POST http://127.0.0.1:8000/tools/

header

Content-Type ：multipart/form-data

body

  • username：用户名
  • tname ： 文件名
  • myfile ： 文件
  • main_file 入口文件
  • main_method 入口方法
  • params ： 参数
  • describe ： 描述

2.工具删除

  • 1.Delete http://127.0.0.1:8000/tools/？tname=工具名
  • params：tname ： 工具名

3.个人工具查询

  • 1.GET http://127.0.0.1:8000/tools/

return：

    '工具序号': id,
    '工具名': tname,
    '工具上传时间': tup_date,
    '工具更新时间':update_time
    '运行次数':run_count

4.条件工具查询

  • 1.POST http://127.0.0.1:8000/tools/toolsquery/

return：

    '工具序号': id,
    '工具名': tname,
    '工具上传时间': tup_date,
    '工具入口文件地址': main_file,
    '工具入口方法': main_method,
    '工具参数':params,
    '工具描述':describe
    '工具更新时间':update_time
    '运行次数':run_count

5.工具调用

  • GET http://127.0.0.1:8000/tools/toolsuse/?id=2

params id：XX

return：

        '工具序号': id,
        '工具名': tname,
        '工具上传时间': tup_date,
        '工具入口文件地址': main_file,
        '工具入口方法': main_method,
        '工具参数': params,
        '工具描述': describe

6.工具占比数量

GET  http://127.0.0.1:8000/tools/toolspercent/

7.用户上传排名

POST  http://127.0.0.1:8000/tools/toolspercent/

order_way  1正序  -1倒序

8.工具总数

GET http://127.0.0.1:8000/tools/toolstotal/

9.工具编辑查询

GET http://127.0.0.1:8000/tools/toolsEditor/?id=2

return

    '工具序号': id,
    '工具名':tname,
    '工具上传时间':tup_date,
    '工具入口文件地址':main_file,
    '工具入口方法': main_method,
    '工具参数':params,
    '工具描述':describe
    '工具最近更新时间':update_time
    '上传文件的窗口显示内容':upload_file

10.工具编辑修改

POST http://127.0.0.1:8000/tools/toolsEditor/

传给我的参数

        id:"工具id"
        username:"当前用户的用户名"
        tname:"文件名"
        main_file:"入口文件" 
        main_method:"入口方法" 
        params:"参数" 
        describe:"工具描述"
        myfile:"上传的工具压缩包"
