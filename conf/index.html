通讯录

注册接口

url:/confapp/contact/registre/
方法:post
参数:
weixin_id:微信id 【必填】
phone:手机号【必填】
head_picture:头像【必填】

返回信息：
{
    "msg":"success",
    "code":1,
    "data":"",
}

首页搜索功能

url:/confapp/contact/search/
方法:get
参数:
id:用户id【必填】
info: 可以是电话、手机号、部门信息【必填】

返回信息：
如果搜索信息是姓名/手机号，返回信息只有name和phone,group和corporate信息为空；
如果搜索的信息是组，返回信息只有group有信息，name/phone/corporate信息为空
同理搜搜企业信息也是如此
{
    "msg": "success",
    "code": 1,
    "data":{
        "person":[{     
            "id":1,
            "name":"张三",
            "phone":"15268117291"，
            "img":"http",}]
        "group":[{
            "id":1,
            "name":"组1",
            "children":[{
                "id":1,
                "name":"张三",
                "img":"http",
                "phone":"15268117291"
        }],
        "corporate":[{
            "id":1,
            "name":"华为组",
            "children":[{
                "id":1,
                "name": "张三",
                "img":"http",
                "phone":"15268117291"          
        }]
    }
}

发起会议

url:/confapp/contact/create_conf/
方法:post
参数:
id:用户id 【必填】
attendee:与会人,[{"phone": "15268117291","name": "孙博",}] 【必填】
theme: 会议主题 默认是发起者的会议【选填】
password:密码【选填】
chairman:主席位 即会议发起人格式同与会人【必填】
type: 1 立即会议 2 预约会议【选填 默认是1】
time: 预约会议开始时间 格式为2020-03-03 19:30:36【选填 预约会议必填】

返回信息:
{
    "msg": "success",
    "code": 1,
    "data":"",
}

开麦闭麦

url:/conapp/control/mike/
方法:post
参数:
id:用户id【必填】
confid:会议id【必填】
attendee:成员[{"name":"张三", "phone":"15268117291"}] 选传 指定人员开麦闭麦【选填】
type: 1全部闭麦 2全部开麦 【必填】
ex_chariman: True 除主席和发言人全部闭麦【选填】
同时存在ex_chariman优先级大于id

返回信息:
{
    "msg": "success",
    "code": 1,
    "data": ""
}
